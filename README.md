# projeto_ia_manutencao_veicular
Projeto de Machine Learning que prevê a manutenção de veículos.

* Desenvolvi um sistema de Manutenção Preditiva utilizando Python para classificar o risco de falha veicular.


* Implementei o algoritmo de Machine Learning Decision Tree Classifier (Classificação) para prever as classes: Risco Alto (1) ou Risco Baixo (0).


* Apliquei técnicas de Pré-processamento de Dados, incluindo Imputação (Moda/Mediana) e Codificação (One-Hot Encoding) para preparar as 8 variáveis de diagnóstico.


* Criei uma interface de usuário (Dashboard) usando Streamlit para deploy do modelo, incluindo um módulo de Explicabilidade (XAI) que fornece o diagnóstico do fator crítico e a solução de reparo ao usuário.

* Tecnologias
Python, Pandas, Scikit-learn, Streamlit, Git/GitHub.
